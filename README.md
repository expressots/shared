## Install

```bash
npm i @expressots/shared
```

## What This Package Does

Shared foundation for the ExpressoTS ecosystem: common TypeScript interfaces, configuration types, environment utilities, and helpers that `@expressots/core`, `@expressots/adapter-express`, and other packages depend on. Most applications consume it indirectly through the core package.
